# Security Policy

LaunchGuard RN is a static audit helper for React Native and Expo apps.

## Reporting security issues

If you find a security issue in this project, please open a GitHub issue with clear reproduction steps.

## Important note

This tool helps detect common risks, but it does not replace a professional security audit.

Always manually review:

- backend permissions
- database rules
- environment variables
- production API keys
- authentication logic
- app store privacy requirements
